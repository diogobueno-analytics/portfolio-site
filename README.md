### 🚀 Portal de Portfólio | Data & BI

Este projeto crio um portal para centralizar meus estudos e projetos em **Business Intelligence e Ciência de Dados**, o site foi desenvolvido com **Python** e **Streamlit**.

Como meu foco profissional não é o desenvolvimento web, utilizei fortemente ferramentas de **IA Generativa (Gemini e Claude)** como aliadas estratégicas. Através de **Prompt Engineering**, transformei minha ideia de site, investindo diversas horas de trabalho no refinamento da interface para garantir uma experiência amigável.
